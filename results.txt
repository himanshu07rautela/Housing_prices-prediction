Files Included:
- housing.csv: The dataset used for training and evaluation.
- lregression.ipynb: Jupyter Notebook containing the entire workflow from data loading to model evaluation.

How to Use:
1) Clone the repository.
2) Install the necessary libraries: pandas, numpy, matplotlib, seaborn, scikit-learn.
3) Open housing.ipynb in Jupyter Notebook and run the cells to see the analysis and results.
